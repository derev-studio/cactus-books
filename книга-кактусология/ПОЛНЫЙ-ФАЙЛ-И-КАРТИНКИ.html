Где полный файл книги и как показать картинки на сайте

Проблема

На странице книги нет картинок — потому что в этой папке нет папки media/ с картинками. Картинки берутся из исходного Word-файла книги.

Шаг 1. Найти у себя полный файл книги с картинками

Файл называется: Кактусология_полная_с_картинками.docx

Он может быть в Документах, на Рабочем столе, в Загрузках или там, где вы храните исходники книги.

Шаг 2. Скопировать этот файл в папку книги

Папка: cactus/книга-кактусология/

Рядом с файлами book-read.html и build_book_with_images.py должен лежать файл Кактусология_полная_с_картинками.docx.

Шаг 3. Запустить скрипт

Откройте терминал, зайдите в папку книга-кактусология и выполните:

python3 build_book_with_images.py

Скрипт извлечёт все картинки из docx в папку media/ и пересоберёт book-read.html.

Шаг 4. Открыть книгу в браузере

Файл: книга-кактусология/book-read.html

Лучше открыть через браузер (Chrome/Safari): правый клик по book-read.html → «Открыть с помощью» → браузер.

Если файла Кактусология_полная_с_картинками.docx нет

Тогда картинки на сайте показать нельзя — они есть только внутри этого Word-документа. Нужно снова положить сюда полный docx и запустить build_book_with_images.py.

Полный путь к папке (куда класть docx): .../Documents/cactus/книга-кактусология/
