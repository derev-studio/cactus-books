# Derev Studio — cactus-books

Личный сайт: кактусы, книги, фотография. Галерея, заметки, комната отдыха, Хранитель (AI-собеседник), творческая комната.

Одностраничное приложение на HTML, CSS и JavaScript. Многоязычность (глобус в шапке), тёмная/светлая атмосфера.

---

## Сайт

**https://derev-studio.github.io/cactus-books/**

---

## Структура

- **index.html** — главная: галерея, блок «Обо мне».
- **stories.html** — заметки/рассказы (из `data/stories.json`).
- **relax.html** — комната отдыха (дыхание, таймер, звук).
- **draw.html** — творческая комната (рисование).
- **guardian.html** — Хранитель (чат с AI).
- **identifier.html** — Определитель кактусов: загрузка фото → вид, ареал на карте, описание, факты, уход (Groq Vision).
- **derev-studio.html** — страница «Обо мне» / о проекте.
- **admin.html** — админка загрузки контента (noindex; для своих нужд).

Галерея подгружает `data/gallery.json` и `data/artworks.json`. Изображения — из репозитория или `img/`.

Определитель кактусов: на странице identifier.html пользователь загружает фото → запрос уходит в **твой** Cloudflare Worker (`POST /v1/vision`) → Groq Vision возвращает JSON с названием, ареалом, описанием, фактами и уходом. Карта ареала — OpenStreetMap (Leaflet).

**Твой сайт — свои ключи.** Чат и определитель работают через **твой** воркер и **твой** API-ключи (в `config.js` указываешь только URL воркера; ключи задаются в Cloudflare Worker Variables). В репозитории нет секретов. Пошаговая настройка — в файле **НАСТРОЙКА.md**.

## Worker (Cloudflare)

В `cloudflare-worker.js` настроены маршруты:
- `POST /v1/chat/completions` — чат Groq (Llama 3.3 70B).
- `POST /v1/image` — генерация картинок (Cloudflare AI).
- `POST /v1/vision` — определитель кактусов: тело `{ "image": "base64...", "mime": "image/jpeg" }` → Groq Vision (Llama 4 Scout), ответ — JSON с названием, ареалом, описанием, фактами, уходом.

Переменные: `GROQ_KEY`, AI binding для картинок. Код воркера размещаешь в **своём** аккаунте Cloudflare; URL воркера прописываешь в **config.js**. Подробно — **НАСТРОЙКА.md**.
