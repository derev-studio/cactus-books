# Кактусярий — мой сайт о кактусах

Личный проект: кактусы, книги, фотография. Галерея, заметки, комната отдыха, Хранитель (AI), определитель кактусов по фото.

Статический сайт (HTML, CSS, JavaScript). Многоязычность, тёмная/светлая тема.

---

## Важно: отдельно от проекта Даши

Этот репозиторий — **твой самостоятельный проект**. Он не связан с репозиторием **derev-studio/cactus-books** (проект Даши). Данные галереи и админка работают с **твоим** репо: в **config.js** укажи `githubOwner` и `githubRepo`. Ключи и воркер — тоже свои (см. НАСТРОЙКА.md).

---

## Структура

- **index.html** — галерея и старт.
- **home.html** — главная.
- **about.html** — о проекте.
- **stories.html** — рассказы (`data/stories.json`).
- **relax.html** — комната отдыха.
- **draw.html** — рисование.
- **guardian.html** — Хранитель (чат с AI).
- **identifier.html** — определитель кактусов по фото (OpenRouter/Groq через твой воркер).
- **admin.html** — админка загрузки контента (noindex).

Галерея: `data/artworks.json` и при заданном `githubOwner` — данные с твоего GitHub. Воркер и ключи — в **config.js** (URL) и в Cloudflare Worker Variables.

---

## Локальный запуск

Открой `index.html` в браузере или: `npx serve .`

---

## Деплой (GitHub Pages)

Ветка `main`, корень репо = корень сайта. В **config.js** задай свой `githubOwner` и `githubRepo` — тогда галерея и админка будут с твоего репо. Подробно — **НАСТРОЙКА.md**.
