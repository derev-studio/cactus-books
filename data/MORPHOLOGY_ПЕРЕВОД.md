# Морфология: перевод на русский (по плану с Gemini)

## Что сделано сейчас

- В карточке вида блок **«Морфология»** показывает текст на русском, если в данных есть поля `morphology_stem_ru`, `morphology_spines_ru`, `morphology_flower_ru`, `morphology_fruit_ru`. Если их нет — показывается английский текст из Wikipedia (`morphology_stem` и т.д.).
- Под блоком морфологии добавлена ссылка **«Прочитать в переводе (Google)»**: по клику открывается Google Переводчик с подставленным текстом морфологии. Так можно сразу прочитать перевод без API и без правок в данных.

## Как потом добавить русский текст в данные (пакетный перевод)

1. **Поля в JSON:** для каждого вида в `data/species/*.json` можно добавить поля `morphology_stem_ru`, `morphology_spines_ru`, `morphology_flower_ru`, `morphology_fruit_ru`. Сайт уже умеет их показывать в приоритете над английским.

2. **Скрипт перевода (когда будет время):**
   - Обходить все `data/species/*.json`, для каждого вида смотреть `morphology_stem` (и др.) без суффикса `_ru`.
   - Отправлять текст в сервис перевода (например [LibreTranslate](https://libretranslate.com/) — бесплатный API, или Google Cloud Translation при наличии ключа).
   - Записывать результат в `morphology_stem_ru` и т.д., сохранять JSON.
   - Делать паузу между запросами, чтобы не перегружать API.
   - После скрипта — вычитка ключевых видов вручную.

3. **Ручной перевод:** при добавлении или правке вида можно сразу заполнять `*_ru` поля — они будут отображаться на сайте.
