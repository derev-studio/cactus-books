# На чём остановились

**Систематика кактусов** в целом готова: дерево, карточки родов и видов, URL вида (`?genus=...&species=...`), перелинковка синонимов, «Смотрите также», фото и морфология из Wikipedia где есть, благодарности и источники.

**Морфология:**
- Язык по браузеру (ru / uk / en): если в данных есть поля `morphology_stem_ru`, `morphology_stem_uk` и т.д. — показываются они, иначе английский. Перехода на Google нет, всё на тёмном сайте.
- **Сделано:** скрипт `scripts/translate_morphology_to_ru.py` — переводит морфологию на русский. **Ничего ставить не нужно** (только встроенный Python, без pip и venv). MyMemory API, паузы 3–4 сек, каждые 25 запросов — 90 сек. Команды:
  ```bash
  cd /Users/alexanderermolovich/Documents/cactus
  python3 -u scripts/translate_morphology_to_ru.py
  ```
  Уже переведённое пропускается. Если «Переведено: 0» — лимит API, запусти завтра.

**Украинский:** скрипт `scripts/translate_morphology_to_uk.py` — то же самое, но пишет в поля `morphology_*_uk`. При языке браузера «украинский» сайт покажет украинский текст и подписи (Квітка, Плід). Команда: `python3 -u scripts/translate_morphology_to_uk.py` (сначала `cd` в папку cactus).

Инструкция: `data/MORPHOLOGY_ПЕРЕВОД.md`.

**Картинки:** часть видов без фото (396) — в en.wikipedia нет подходящих статей. Список: `data/SPECIES_WITHOUT_PHOTO.txt`. Решать позже: другой источник или ручные ссылки.

**Правило:** после изменений делать `git add -A`, `git commit`, `git push origin main`, чтобы всё было на сайте.
