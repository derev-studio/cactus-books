# Как отправить изменения на сайт

- **Автоматически (включено):** в проекте есть правило для Cursor — после любых правок помощник **сам** делает коммит и пуш. Тебе достаточно нажать синюю кнопку (Accept), чтобы принять изменения: они сразу уйдут на сайт. Работает в любом чате.
- **Вручную:** дважды щёлкни по файлу **ОТПРАВИТЬ-НА-САЙТ.command** в папке cactus — он сохранит все изменения и отправит на GitHub.

Сайт: https://derev-studio.github.io/cactus-books/
