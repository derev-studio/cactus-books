# Настройка сайта и воркеров

## 1. Ключи и токены (безопасность)

**В репозитории нет и не должно быть реальных ключей.** Все секреты задаются только в окружении.

- **Cloudflare Worker (OpenRouter + DeepL)** — файл `worker-openrouter.js`  
  В настройках Worker → Variables and Secrets добавь:
  - `OPENROUTER_API_KEY` (Secret) — ключ с [OpenRouter](https://openrouter.ai)
  - `DEEPL_API_KEY` (Secret) — ключ [DeepL Free API](https://www.deepl.com/pro-api)

- **Cloudflare Worker (Groq)** — файл `cloudflare-worker.js`  
  В настройках Worker добавь:
  - `GROQ_KEY` (Secret) — ключ с [Groq Console](https://console.groq.com)
  - `DEEPL_API_KEY` (Secret) — для перевода книги  
  И binding **AI** для генерации картинок.

- **GitHub токен для админки** — вводится вручную в поле на странице `admin.html` и хранится только в `sessionStorage` браузера. В код не попадает.

## 2. URL воркера в config.js

После деплоя воркера скопируй его URL (например `https://твой-воркер.твой-аккаунт.workers.dev`) и вставь в **config.js** в поле `workerUrl` (без слэша в конце).

## 3. GitHub Pages

Репозиторий: `derev-studio/cactus-books` (или твой fork).  
В настройках репозитория: Settings → Pages → Source: ветка `main`, папка `/ (root)`.  
Сайт будет доступен по адресу: `https://derev-studio.github.io/cactus-books/`.
