# Перевод морфологии

Скрипт работает без установки библиотек: только встроенный Python, два API по очереди (Free Translate API + MyMemory). Запуск:

  cd /Users/alexanderermolovich/Documents/cactus
  python3 -u scripts/translate_morphology_to_ru.py

Если захочешь спросить у Gemini про другие варианты, можно спросить так: «Какой бесплатный API перевода с английского на русский можно вызывать из Python без ключа и без установки библиотек (только urllib)?»
