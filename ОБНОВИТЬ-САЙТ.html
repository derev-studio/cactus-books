Обновить сайт в интернете

Это твой проект (Кактусярий). Обновляй свой репозиторий: в папке проекта открой терминал и выполни
git add . && git commit -m "Update" && git push
Сайт обновится через 1–2 минуты (GitHub Pages). Репозиторий Даши этим не затронут.
Ниже — инструкция для проекта Даши (репозиторий derev-studio/cactus-books). Если ты обновляешь её сайт (копируешь файлы в её репо) — используй способы 1 или 2. Если работаешь только со своим сайтом — ими не пользуйся.

Способ 1: Через Cursor (для обновления сайта Даши)

Если один раз настроишь — потом только копируешь файлы и жмёшь одну кнопку в Cursor.

Один раз: Склонировать репозиторий в Cursor.
Нажми Ctrl+Shift+P (или Cmd+Shift+P на Mac) → введи Git: Clone → выбери эту команду. Вставь ссылку: https://github.com/derev-studio/cactus-books → выбери папку (например Рабочий стол). Когда Cursor спросит логин и пароль: пароль — это твой токен GitHub (PAT), который ты сохранял для кактуса. Логин можно оставить derev-studio.
Если при Push пишет «нет разрешения» или другой аккаунт — заново клонируй, но вставь ссылку с токеном: https://ТВОЙ_ТОКЕН@github.com/derev-studio/cactus-books (подставь свой PAT вместо ТВОЙ_ТОКЕН). Тогда Push будет от твоего аккаунта.
Каждый раз при обновлении:
1) Скопируй из папки Документы → cactus всё нужное (или хотя бы site-header.css, styles.css, book.html) в папку cactus-books → cactus (ту, что появилась после клонирования). Замени файлы.
2) В Cursor открой эту папку: File → Open Folder → выбери cactus-books.
3) Слева нажми иконку ветки (Source Control). Увидишь список изменённых файлов. Нажми + рядом с «Changes», чтобы добавить все. Внизу введи сообщение, например Update. Нажми синюю кнопку Sync или Push.
После Push подожди 1–2 минуты. Открой сайт и нажми Ctrl+Shift+R (Cmd+Shift+R на Mac) — появятся новые кнопки.

Способ 2: Через браузер (для обновления сайта Даши)

⚠️ Одна только ссылка сайт не обновляет. Нужно загрузить файлы и нажать Commit.
Шаг 1. Открыть папку cactus на GitHub
Шаг 2. На странице GitHub нажми «Add file» → «Upload files».
Шаг 3. Из папки Документы → cactus перетащи в браузер три файла: site-header.css, styles.css, book.html → внизу нажми «Commit changes».
Через 1–2 минуты открой сайт и нажми Ctrl+Shift+R — тогда появятся новые кнопки.
